Tags

HTML tags are composed of an opening tag, content, and a closing tag.
The opening tag marks the beginning of an element, and the closing tag marks the end.
The content is the information or structure that falls between the opening and closing tags.

Self-closing-tag

1. br

2. hr

3. input

Attributes

HTML attributes provide additional information about HTML elements and are always specified in the start tag.
They usually come in name/value pairs like name="value".
