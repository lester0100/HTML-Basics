This folder contains four small HTML practice projects. Each file is useful for learning, experimenting, or reference.

## Projects included
- **1-first-webpage.html**: A simple landing page that introduces HTML, CSS, and JavaScript with basic headings, paragraphs, and buttons.

- **2-tags-and-attributes.html**: Shows and explains HTML tags and attributes, including self-closing tags and a basic input element.

- **3-text-formatting.html**: Gives examples of text tags like bold, strong, italic, emphasis, underline, subscript, superscript, deleted text, and the old font tag (for history).

- **4-Href-challenge/**: A small multi-page example that shows how links and navigation work:
  - index.html: a landing page with links to jump to the top or bottom, plus links to subpages and an outside resource.
  - Tags/tag_page.html: a short overview of HTML tags and self-closing tags.
  - attribute_page.html: simple examples of attributes and an input field.
  - list_and_tables.html: examples of ordered and unordered lists, plus a simple table for a menu or pricing demo.
